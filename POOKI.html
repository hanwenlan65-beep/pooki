POOKI MINI BRAND GUIDE

温柔高级，从“克制 + 呼吸感”开始

这份小型VI用于你目前阶段：小红书内容、贴纸与包装、说明卡、展签与买手店陈列。 目标是把“轻松”做得高级，把“设计感”做得不疏离。

温柔 克制 呼吸感 完成度 外松内严
一句话定位：pooki 噗物｜轻松，但讲究。

展示版 Logo（Display）

用于：展签 / 海报 / 包装正面 / 贴纸主视觉 / 封面大留白角落

日常版 Wordmark（推荐：pooki）

用于：头像小尺寸 / 水印 / 产品底刻 / 说明卡 / 发货标签

1. 品牌调性

关键词

温柔 克制 呼吸感 完成度 外松内严

禁用气质

  • 可爱幼稚 / 过度卖萌
  • 花哨装饰 / 赛博酷 / 过强街头硬核
  • 信息太满、排版太吵
核心原则：少即是多。先留白，再说话。

2. 品牌语言

推荐句式

  • 轻松，但讲究。
  • 把设计放回手边。
  • 小物，也要刚刚好。

禁用词（会拉低高级感）

  • 绝绝子 / 爆款 / 必入 / 巨好用 / 仙女 / 过度感叹号
写法建议：少形容词，多事实（比例、边缘、结构、手感、稳定性）。

3. Logo系统

A. 展示版（Display Logo）

  • 两行块面（POO / KI）
  • 适合远距离识别与陈列

B. 日常版（Wordmark）

  • 推荐：小写 pooki（更温柔高级）
  • 用于小尺寸与细节位置

C. 小标（Stamp）

  • 单字母 P 或 PK 方章（可后补）

4. Logo使用禁忌

  • 不要加渐变、金属镜面、强阴影特效
  • 不要把 Logo 贴着边放（破坏呼吸感）
  • 小尺寸时不要用展示版强撑，换日常版
提示：展示版适合“画面主角”；日常版适合“内容配角”。

5. 安全留白（Safety Clear Space）

以 O 的内孔直径 d 作为单位。

最小安全区：四周 ≥ 1d（展签/包装正面建议 ≥ 1.5d）
单位定义：d = O 内孔直径

原则：宁可 Logo 小一点，也不要挤满画面。

6. 最小使用尺寸（Minimum Size）

展示版（两行块面）

  • 屏幕端最小宽度：160 px
  • 印刷端最小宽度：25 mm

日常版（pooki）

  • 屏幕端最小宽度：90 px
  • 印刷端最小宽度：15 mm
规则：小于阈值 → 直接换日常版。

7. 颜色系统（少而准）

主色

  • 柔黑：避免纯黑（#000），更温柔、耐看
  • 奶白/雾白背景：避免刺眼纯白

允许的配色组合

  • 柔黑 on 奶白（默认）
  • 奶白反白 on 柔黑（贴纸/箱贴/展牌）
  • 柔黑 on 暖灰（质感图很稳）

8. 色彩禁忌

  • 高饱和彩色大面积抢戏
  • 强渐变、强发光、金属镜面特效
  • 同一张图里出现过多颜色（>3）
一句话：颜色越少，越像买手店。

9. 排版与文字

层级原则

  • 标题少字，尽量 ≤ 8 个字
  • 留白优先，信息第二
  • 少感叹号、少热词

推荐标题示例

桌边小托盘 01 窗台花盆 02 一盏不打扰的小灯

10. 文案语气（高级感来源）

推荐

  • 更稳定 / 更顺手 / 更耐看 / 更通透 / 更克制
  • “我改了哪里”“为什么这样更舒服”

避免

  • “好高级/好有设计感”这类自夸式表达
原则：讲事实，观众自己得出“高级”。

11. 摄影与画面规则

背景

  • 奶白、暖灰、木色、石材（低饱和、干净）
  • 避免杂乱桌面、强纹理背景抢戏

光

  • 自然光/柔光，阴影干净
  • 避免强反光与硬影

12. 构图模板（建议固定）

  • 画面留白 ≥ 60%
  • 产品占画面 20%～35%
  • Logo 放角落：右下/左下（不要压在产品上）
推荐：像画廊陈列一样拍，而不是摆摊展示。

13. 材质与工艺气质（3D打印）

  • 统一哑光（高级感最稳）
  • 允许轻微砂纹/细磨砂
  • 尽量减少“亮面塑料感”和明显层纹
优先级：先把表面质感做统一，再谈配色。

14. 工艺表达方式

  • 少装饰，多“结构的小巧思”
  • 细节呈现：倒角/接触面/稳定性/重心
  • 用对比图讲“改动带来的舒适”

15. 小红书封面模板

  • 奶白底 + 产品大图 + 角落小 Logo
  • 标题 ≤ 8 字，尽量一行
封面结构： [大留白] [产品] (角落) Logo + 小标题

16. 说明卡模板（随包裹）

  • 正面：产品名 + 一句话
  • 背面：材质/使用方法/保养 + 署名
正面： 桌边小托盘 01 轻松，但讲究。 背面： 材质： 使用： 保养： Designed by pooki

17. 定稿检查清单（每次出图前）

  • 留白够不够？（安全区 ≥ 1d）
  • 缩小到头像会不会糊成黑块？
  • 文案是否能删掉一半词？
  • 颜色是否 ≤ 3？是否低饱和？
  • 是否避免了渐变/金属/强阴影？
  • 产品表面质感是否统一（哑光优先）？
  • 主视觉是否“像画廊陈列”而不吵？
一句话：少一点更高级，松一点更温柔。
